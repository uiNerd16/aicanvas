# Andromeda

A sci-fi / blueprint design system for AI Canvas. Near-monochrome, JetBrains Mono, a single turquoise accent, hairline corner markers.

## Editions

**Andromeda v1 (free, MIT).** The components and tokens in this folder are free to use, forever. Use them in anything, including commercial work.

**Andromeda v2 (premium).** The full design brain (the deep rules and the build workflow), the template library, and one-command bulk install. Available to premium subscribers.

Learn more: https://aicanvas.me/andromeda

## Development

This public repository is Andromeda **v1**, frozen and free under MIT. From here on, all new Andromeda work is developed in the private AI Canvas repository and is not published here. New components stay free to use, but their source lives with the platform rather than in this public repo. Templates, the design brain, and bulk install are premium.
