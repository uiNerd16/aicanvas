# OG fonts

Manrope (Medium 500, ExtraBold 800), TrueType, fetched from Google Fonts.
Used only server-side by the `/og/*` image routes (`next/og` ImageResponse
requires raw ttf/otf data; the woff2 served to browsers is not supported).
